Github + Hexo搭建自己的博客——配置主题

发表于 2019-10-26 分类于 blog 2.4k

前言

前两篇文章我们已经把环境配置好并且已经部署到github上，但是还没有内容。而且hexo默认的主题感觉并不是很好看。本文将说一说如何配置主题，如何创建博文，写好后又怎么发布!

选择一个自己喜欢的主题

点我选择主题 (能不能进去看缘分，当然你也可以在github上直接搜索)

更换主题

本文主要介绍目前用得比较多的next主题的配置，其他主题的配置大同小异。

  1. 下载主题: 进入Blog文件夹下的theme文件夹，然后打开终端，输入git clone https://github.com/theme-next/hexo-theme-next。
  2. 应用主题:下载完成后，进入Blog文件夹下的_config.yml,修改themes的值为hexo-theme-next。
